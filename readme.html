Car Lease Payment Calculator app installation

This application calculates the monthly fee when you lease a car.

Its installation is very easy. All you need to do is to copy ui.R and server.r files to a local directory. Next, change the directory in R Gui or RStudio to this directory, then run a command:

runApp()

The Web page will have the input panel on the left, and the output on the right side. The inputs are typical lease terms, such as the length of a lease, interest rate etc. The output is the monthly payment.
